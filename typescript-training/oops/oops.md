## Class
Class is a data type or template that is going to provide a structured way to define all the data,  methods and objects together in one place. 

## Object
Object is a new instance of the class. (New Xerox copy of the class. )

## Constructor 
Constructor is a default method created by TypeScript every time we are going to create a new class. The constructor is going to initialize all the properties of the class whenever we are going to create an object.
