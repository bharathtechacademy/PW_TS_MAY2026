# What is a program? 
A program is all about a set of instructions that we are going to share with our computer to complete a specific task. 

Example: 
1. Launch the "Chrome" browser. 
2. Enter the URL " www.google.com "
3. Enter the keyword "Playwright" in the "Google search" text box. 
4. Press Enter button and capture the test results. 

# What is Instruction ?
Instruction is a combination of data plus respective action. 

Example:
1. Launch the "Chrome" browser. 

data: Chrome
action : Launch
